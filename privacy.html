5. Data Security

We implement industry-standard security measures to protect your data:

  • Encryption in transit: All data transmitted via HTTPS/TLS encryption
  • Row-Level Security (RLS): Database-level access controls ensure users can only access their own data
  • API key security: All AI API keys stored as server-side environment variables, never exposed client-side
  • Authentication: Secure authentication via Supabase Auth with email verification
  • Payment security: We never store full card numbers — Stripe handles all payment data under PCI-DSS compliance
  • Access controls: Administrative access limited to authorized personnel only

Despite these measures, no security system is impenetrable. In the event of a data breach that affects your personal information, we will notify you as required by applicable law.

Under the Texas Data Privacy and Security Act (TDPSA) and applicable federal law, you have the following rights:

  • Right to Correction: Request correction of inaccurate personal data
  • Right to Deletion: Request deletion of your personal data
  • Right to Data Portability: Request your data in a machine-readable format
  • Right to Opt Out of Sale: We do not sell your data, but you can confirm this at any time
  • Right to Non-Discrimination: We will not discriminate against you for exercising your privacy rights

To exercise any of these rights, email [email protected] with "Privacy Request" in the subject line. We will respond within 45 days as required by law.
